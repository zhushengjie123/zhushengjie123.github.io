# 片段

即多根节点组件。

## 片段介绍

`vue2` 中，一个组件中只能有一个根节点（所以只能用 div 将多个节点包裹）。

```html
<!-- Layout.vue -->
<template>
  <div>
    <header>...</header>
    <main>...</main>
    <footer>...</footer>
  </div>
</template>
```

在 `vue3` 中组件可以包含多个根节点。

但是，这要求开发者显式定义 `attribute` 应该分布在哪里，具体参考非 prop 的 attribute 的继承。

```html
<!-- Layout.vue -->
<template>
  <header>...</header>
  <main v-bind="$attrs">...</main><!--被指定传递 attribute 的节点 -->
  <footer>...</footer>
</template>
```
